# Japanese Word Frequency

A Python project that utilizes MeCab to analyze a SRT file and return the frequency of the words.

## Setup

1. Install python 3.12

```shell
brew install python@3.12
```

2. Install pipenv

```shell
pip install pipenv
```

3. Install dependencies

```shell
pipenv install
```

4. Start pipenv

```shell
pipenv shell
```
